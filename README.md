# 项目简介

## 核心功能
- 区分旅游项目的核心类别
  - 自然风光类
  - 文化体验类
  - 休闲度假类
  - 康养旅游类
  - 重大投资类
- 项目状态
  - 筹备中
  - 新开工
  - 续建
  - 竣工投产
  - 暂停建设
- 总投资
  - 项目总资金规模（单位：万元，保留 2 位小数）
- 项目推荐（优选项目）
  - 地图查看项目位置
- 项目审核（通过 / 驳回）
- 视频上传、播放
- 个人信息管理
  - 个人信息预览、修改
  - 用户密码修改
  - 用户登录、退出

## 适用场景
- 旅游项目管理和规划
- 旅游项目投资决策
- 旅游项目进度监控
- 旅游项目信息展示

## 技术特点
- 基于 Java 实现，具有良好的跨平台性
- 源码、数据库文件、部署教程及功能演示视频一应俱全
- 提供直观的项目状态和投资信息展示
- 支持地图定位查看项目位置
- 具备个人信息管理和安全机制
